# CLAUDE.md - Net Worth Tracker (Lean)

## Project Overview
Net Worth Tracker is a Next.js app for Italian investors to track net worth, assets, cashflow, dividends, performance metrics, and long-term planning with Firebase.

## Known Issues (Active)
- FX conversion depends on Frankfurter API availability; cache fallback (24h TTL) is used on failure ‚ÄĒ pre-migration non-EUR assets (USD, CHF, etc.) without `currentPriceEur` will show native price as EUR until first price update. GBp assets are safe: the pence‚ÜíGBP fallback (`currentPrice / 100`) prevents the 100√ó inflation even without `currentPriceEur`.
- Demo account requires manual Firebase setup (create user, populate Firestore with realistic fake data, set three env vars)
